# Salvage

Reusable inputs lifted from two retired from-scratch apps before they were
archived + removed locally. See [`../WEDGE.md`](../WEDGE.md) for how they feed
the wedge.

- **`from-posthive/`** — PRD v2, design system, UI spec. The product/brand
  *thinking* (funnel logic, design tokens, security-UX framing) is reusable;
  posthive's Next.js app code is not (Postiz supersedes it).
- **`from-v3nex/FEATURES.md`** — feature/billing/provider inventory; v3nex's
  code is redundant with Postiz.

**Full source of both is preserved** in their archived GitHub repos
(`rodriguescarson/posthive`, `rodriguescarson/v3nex`) — archive is read-only,
not deleted, so anything not copied here is still recoverable there.
